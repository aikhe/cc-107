Research & Components

Comprehensive breakdown of computing architecture components including hardware, software, dataware, and liveware.

Hardware

GPU

Specialized processor designed to accelerate graphics rendering and parallel processing tasks.

CPU

Central Processing Unit, the primary component representing the "brain" of the computer.

RAM

Random Access Memory provides high-speed temporary storage for running applications.

Storage

Persistent data storage solutions including HDDs and SSDs for long-term file retention.

Microcontrollers

Compact integrated circuits like Raspberry Pi used for embedded systems and DIY projects.

Software

Windows

Dominant desktop OS by Microsoft, known for broad compatibility and gaming.

macOS

Apple's Unix-based OS, renowned for stability, creative workflows, and ecosystem ecosystem.

Linux

Open-source OS kernel powering everything from servers to supercomputers and Android.

Android

Mobile operating system developed by Google, based on a modified Linux kernel.

iOS

Apple's mobile OS powering iPhone, known for its security and polished UX.

Dataware

Google BigQuery

Serverless, highly scalable, and cost-effective multi-cloud data warehouse designed for business agility.

Snowflake

Cloud computing-based data cloud company that offers a cloud-based data storage and analytics service.

Amazon Redshift

Fast, fully managed cloud data warehouse that makes it simple and cost-effective to analyze all your data.

Liveware

Programmer

Specialist who writes code to create software applications and systems.

Manager

Oversees the IT infrastructure, projects, and teams to ensure alignment with business goals.

User

The end-user who interacts with the computer systems and applications to perform tasks.
